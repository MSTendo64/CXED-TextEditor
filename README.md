# CXED simple Text editor

**You can use:**
- cxed path/to/file
- cxed (without arguments)

![Using](/screenshots/Using.png)

**Input:**
- Upper / Lower arrows - move along the lines
- tab - Adds 4 spaces
- ctrl + d - Save
- ctrl + x - Exit
